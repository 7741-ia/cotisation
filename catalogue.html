Guide & Catalogue — App de Cotisation

Créateur : Daniel Amisi Awazi — Version 1.0.0

Vue d'ensemble

Cet espace sert de catalogue / manuel d'utilisation pour l'application de gestion des cotisations. Vous y trouverez comment créer un compte, ajouter des membres, enregistrer des paiements, exporter et partager les informations.

Comment utiliser (pas à pas)

  1. Créer un compte : sur la page d'accueil, entrez votre email et un mot de passe puis cliquez S'inscrire.
  2. Compléter le compte : après la création ou à chaque connexion vous êtes redirigé vers la page Compléter votre compte où vous pouvez ajouter les premières personnes et leur montant mensuel.
  3. Tableau de bord : cliquez sur Aller au tableau pour voir vos membres, totaux et historique des paiements. Seuls les membres que vous avez créés s'affichent dans votre tableau (sauf si vous êtes administrateur).
  4. Ajouter un paiement : dans la colonne Actions, cliquez Ajouter pour enregistrer un paiement pour un membre.
  5. Modifier un membre : en tant que créateur du membre (ou admin), cliquez Modifier pour changer le montant total ou la cotisation mensuelle.
  6. Exporter / Importer CSV : utilisez les boutons Exporter CSV et Importer CSV pour sauvegarder ou restaurer des listes de membres. Le CSV contient le total, la mensuelle, payé ce mois et le restant.
  7. Partager (vue publique) : les créateurs peuvent générer un lien « lecture seule » pour montrer l'état des paiements à une autre personne sans lui donner le droit de modifier.

Administration et sécurité

  • Administrateurs : peuvent visualiser et modifier tous les membres. Le mot de passe admin par défaut local est bestking (changez-le en production).
  • Mot de passe oublié : utilisez le lien Mot de passe oublié ? sur la page de connexion pour simuler l'envoi d'un code et réinitialiser votre mot de passe.
  • Données : en mode local toutes les données sont stockées dans localStorage. Si vous utilisez le serveur Node, les données sont persistées dans data.json.

FAQ rapide

Pourquoi je ne vois pas les membres d'un autre utilisateur ?
Chaque compte a son propre tableau. Seuls le créateur (ou un admin) peut modifier/supprimer ses membres.
Comment rendre un lien public ?
Générez un lien de partage depuis le tableau (bouton Partager). Pour que le lien fonctionne partout, il faut persister le token côté serveur ; on peut l'ajouter si vous le souhaitez.
